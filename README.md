# Customer Trend Data Analysis (SQL + Python + Power BI)

## Project Overview
This project analyzes customer shopping behavior using transactional data from **3,900 customer purchases** to identify trends, customer segments, spending patterns, and product preferences. The goal is to help businesses improve marketing strategies, customer engagement, and product performance.

## Business Problem
A retail company wants to understand customer shopping behavior across:
- Demographics
- Product categories
- Discounts & promotions
- Subscription plans
- Shopping seasons
- Payment methods

Main objective:
**How can customer shopping data be used to improve sales, customer loyalty, and marketing strategies?**

## Tools & Technologies
- **Python** (Pandas)
- **SQL** (sql server)
- **Power BI**
- **Jupyter Notebook**

## Project Workflow
### 1. Data Cleaning & Preparation (Python)
- Loaded raw dataset
- Handled missing values
- Standardized column names
- Created age groups
- Feature engineering
- Exported cleaned dataset for SQL analysis

### 2. Data Analysis (SQL)
Performed business queries such as:
- Revenue by gender
- High spending discount users
- Top-rated products
- Shipping type analysis
- Subscription vs non-subscription comparison
- Customer segmentation
- Revenue by age group

### 3. Dashboard (Power BI)
Created interactive dashboard showing:
- Total customers
- Average purchase amount
- Revenue by category
- Subscription insights
- Sales by age group
- Product performance

## Key Insights
- Male customers generated higher revenue than female customers.
- Loyal customers contributed the highest number of purchases.
- Express shipping users had slightly higher average spending.
- Subscription customers showed strong retention.
- Young adults generated the highest revenue.

## Business Recommendations
- Improve loyalty programs for repeat customers.
- Promote subscription benefits.
- Optimize discount strategy.
- Focus on high-performing products.
- Target high-revenue customer age groups.
